Logic — Privacy Policy

Last updated: March 11, 2025

What Logic does

Logic is a Chrome extension that uses the Anthropic Claude API to analyze your thinking through the Concept Engine protocol. You provide your own API key.

Data we collect

We do not collect personal data. Logic stores your settings (API key, preferences) locally in your browser using Chrome's storage API. This data never leaves your device except when communicating with Anthropic's API.

Collective Network (opt-in only)

If you choose to participate in the collective network, Logic shares:

  • The combination of system levels touched by a relation (e.g., "A6+P1+L1")
  • A short description of the relation (max 280 characters)
  • Fidelity scores (C values before and after)
  • An anonymous identifier (random UUID generated on install, not linked to any personal information)

This data is stored on Supabase (PostgreSQL). No names, emails, browsing history, or personal information is collected or transmitted.

API communication

Your messages are sent directly from your browser to Anthropic's API using your own API key. Logic does not proxy, store, or log your conversations. Anthropic's own privacy policy applies to that communication.

Web page analysis

When you choose to analyze a web page, the page content is extracted locally and sent to Anthropic's API as part of your conversation. It is not stored or sent anywhere else.

Third-party services

  • Anthropic API — processes your messages. See Anthropic's Privacy Policy.
  • Supabase (opt-in collective only) — stores anonymous relation data. See Supabase's Privacy Policy.

Data deletion

All local data can be deleted by uninstalling the extension or clearing extension data in Chrome settings. To delete collective network data, contact us.

Contact

For questions about this privacy policy, open an issue on our GitHub repository.
