Every concept is decomposed into Subject, Object, and Relation fidelity, scored against a Sought Value. Any component at zero collapses the entire concept. Partial fidelity is more dangerous than zero.

What Logic does

Decompose

Every input is broken into S, O, R, and VB. If the Sought Value is absent, the system halts until you declare it.

Score

Each component gets a fidelity score from 0 to 1. Below 0.5 means the concept needs reconstruction before use.

Eliminate

Only what survives contact with territory in 3+ contexts is retained. Empty residue is declared NULL, not named.

Audit

Traces where your concepts came from. Inherited language enters the map before the auditing instrument exists.

Collective Intelligence

Share discovered relations between system levels. When multiple users independently find the same connection, it becomes verified territory.

Declare Limits

Unknown territory is marked, never filled with projection. "I don't know" at the map edge is more navigable than confident invention beyond it.

Get started

1

Download the extension from GitHub and load it in Chrome (Developer mode > Load unpacked)

2

Enter your Anthropic API key in Options. You bring your own key — Logic stores nothing on external servers.

3

Click the Logic icon to open the side panel. Start thinking. The system will decompose, score, and audit your concepts.

4

Analyze any web page through the Concept Engine lens. Logic detects who publishes and why, not just what.

Core principles

The navigator is the map — not: the navigator has a map
The map is not the territory — partial fidelity is more dangerous than zero
The territory always wins — when map and territory diverge, update the map
Distortion accumulates by default — reversal requires active energy
Comprehension = elimination — what survives testing, name it. What doesn't, discard it.
